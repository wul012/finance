# 数据字典

## materials.json

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 货币材料或信用载体唯一 ID |
| name | string | 中文名称 |
| category | string | 类型，例如贵金属货币材料、国家信用货币 |
| active_from_year | integer | 开始出现或形成代表性影响的年份，公元前年份使用负数 |
| active_to_year | integer/null | 代表性货币地位结束年份，仍在使用则为 null |
| value_basis | string | 价值基础 |
| strengths | string[] | 主要优势 |
| limits | string[] | 主要局限 |

## assets.json

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 资产唯一 ID |
| name | string | 资产名称 |
| asset_type | string | 资产类型 |
| quote_currency | string | 计价货币或指数类型 |
| unit | string | 价格单位 |
| description | string | 影响因素摘要 |
| related_material_ids | string[] | 关联货币材料 ID |

## events.json

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 事件唯一 ID |
| year | integer | 事件年份 |
| name | string | 事件名称 |
| summary | string | 事件摘要 |
| direct_reason | string | 直接原因 |
| deep_reason | string | 深层原因 |
| affected_assets | string[] | 影响资产 ID，`*` 表示广泛影响 |
| affected_materials | string[] | 影响材料 ID |
| tags | string[] | 事件标签 |
| source | string | 来源或来源类型 |

## prices.csv

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| asset_id | string | 资产 ID |
| year | integer | 观察年份 |
| price | decimal | 价格或指数点位 |
| unit | string | 价格单位 |
| source | string | 数据来源描述 |
| note | string | 观察点说明 |

## solana_research.json

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| overview | string | SOL 与 Solana 网络的定位和价值边界 |
| value_mechanisms | ResearchItem[] | 手续费、质押、流动性和宏观定价机制 |
| catalysts | ResearchItem[] | 可验证的潜在催化剂，不代表价格预测 |
| risks | ResearchItem[] | 市场、网络、供给、活动质量和监管风险 |
| milestones | Milestone[] | SOL 发展过程中的关键事件 |

`ResearchItem` 包含 `title`、`detail` 和 `source`；`Milestone` 包含 `date`、`title`、`impact` 和 `source`。

## SOL 在线市场字段

| 字段 | 类型 | 来源 | 说明 |
| --- | --- | --- | --- |
| observed_at | datetime | CoinGecko | 行情快照 UTC 时点 |
| price_usd | decimal | CoinGecko | SOL 美元价格 |
| market_cap_usd | decimal | CoinGecko | 流通市值 |
| volume_24h_usd | decimal | CoinGecko | 24 小时成交额 |
| change_24h_pct | decimal | CoinGecko | 24 小时价格变化百分比 |
| tvl_usd | decimal | DeFiLlama | Solana 链 DeFi TVL |

在线字段不写入种子数据；生成的报告会固定保存当次观察值与时间戳。
