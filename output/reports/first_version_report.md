## 2. 货币材料演变概览

### 贝壳货币

- 类型：实物货币材料
- 活跃阶段：公元前3000年 至 1200年
- 价值基础：稀缺性、装饰性、可识别性和共同习俗
- 优势：便于识别；适合小规模交换；具有装饰和礼仪价值
- 局限：耐久性和分割性有限；跨区域价值标准不稳定；难以支撑大规模贸易

### 铜与青铜

- 类型：金属货币材料
- 活跃阶段：公元前1200年 至 1900年
- 价值基础：金属耐久性、铸造能力和日常交易便利性
- 优势：适合小额交易；可铸造成统一形制；供应相对充足
- 局限：单位价值较低；大额支付运输成本高；易受工业需求影响

### 白银

- 类型：贵金属货币材料
- 活跃阶段：公元前700年 至 1935年
- 价值基础：稀缺性、可分割性、耐久性和跨区域接受度
- 优势：适合中等面额；历史流通范围广；兼具货币和商品属性
- 局限：供应发现会改变相对价值；与黄金比价长期波动；在现代货币制度中地位下降

### 黄金

- 类型：贵金属货币材料
- 活跃阶段：公元前600年 至 1971年
- 价值基础：高度稀缺、耐腐蚀、可分割、全球认可
- 优势：适合储值；跨政权和跨国界认可度高；危机时期避险属性强
- 局限：供应弹性有限；不适合高频小额支付；金本位会限制货币政策弹性

### 纸币

- 类型：信用货币载体
- 活跃阶段：1024年 至 至今
- 价值基础：发行者信用、可兑换承诺或国家法偿地位
- 优势：携带成本低；适合大规模流通；面额设计灵活
- 局限：依赖发行纪律；滥发会导致通胀；需要制度和法律背书

### 银行存款货币

- 类型：金融机构信用
- 活跃阶段：1600年 至 至今
- 价值基础：银行资产负债表、存款保险和清算体系
- 优势：支付效率高；可与信贷创造结合；适合现代商业活动
- 局限：依赖银行稳健性；可能出现挤兑；需要监管和最后贷款人

### 法定信用货币

- 类型：国家信用货币
- 活跃阶段：1971年 至 至今
- 价值基础：国家信用、税收能力、央行制度和法偿规定
- 优势：货币政策弹性高；适合现代银行体系；可支持信用扩张
- 局限：可能发生通胀或信用危机；汇率受政策和预期影响；跨境信任依赖国家实力

### 电子支付余额

- 类型：数字化信用载体
- 活跃阶段：1990年 至 至今
- 价值基础：账户体系、支付网络、监管许可和用户信任
- 优势：交易速度快；边际成本低；适合线上经济
- 局限：依赖网络和平台；存在隐私与安全问题；跨平台互操作性受限

### 加密资产

- 类型：网络共识资产
- 活跃阶段：2009年 至 至今
- 价值基础：算法稀缺性、网络共识、可验证账本和市场流动性
- 优势：无需传统发行主体；全球可转移；供应规则透明
- 局限：价格波动大；监管不确定；支付和估值共识仍在演化

### 央行数字货币

- 类型：数字化国家信用货币
- 活跃阶段：2020年 至 至今
- 价值基础：央行负债、法偿属性、数字支付基础设施
- 优势：兼具国家信用和数字化效率；有利于支付基础设施升级；可增强货币流通监测
- 局限：需要平衡隐私和合规；会影响商业银行负债结构；跨境使用规则仍在探索

## 5. 第一版结论

1. 货币材料从实物稀缺性逐步转向制度信用和网络信用。
2. 金银等材料的长期价格变化，既受供需影响，也受货币制度、实际利率和避险需求影响。
3. 工业金属更容易体现产业周期，法币汇率更依赖国家信用、贸易结构和政策框架。
4. 加密资产的价格波动具有更强的叙事、流动性和监管敏感性。
